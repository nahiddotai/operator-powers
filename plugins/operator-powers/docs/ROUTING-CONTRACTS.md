# Operator Powers routing contracts

The catalogue can contain adjacent jobs without making them interchangeable. These ownership rules are the tie-breakers used by skill descriptions, local discovery, catalogue search, and collision tests.

| Boundary | Choose the first when | Choose the second when |
|---|---|---|
| Meeting Miner / Customer Insight Synthesizer | One meeting or a meeting-specific comparison is the unit of work. | Patterns across multiple interviews, reviews, surveys, support messages, or mixed customer evidence are the unit of work. |
| Customer Insight Synthesizer / Case Study Builder | The output is a pattern across customers. | The output is one publishable customer or project story. |
| Offer Builder / LLM Council | The user needs to define or improve what is sold. | The user needs to compare or pressure-test genuine choices. |
| Offer Builder / Digital Product Launch Builder | The buyer, promise, package, scope, or price is not yet defined. | A defined product needs a coordinated launch. |
| Landing Page CRO / Offer Builder | The offer remains fixed and the existing page needs to convert more clearly. | The underlying promise, package, audience, scope, or price must change. |
| Landing Page CRO / De-Slop | Conversion structure, proof, CTA, trust, or friction is the job. | Voice and AI-writing tells are the job. |
| Brand System Builder / Voice DNA | Reusable visual identity is the deliverable. | Reusable written voice is the deliverable. |
| Brand System Builder / Carousel or Slideshow | Visual rules must work across many assets. | One finished asset is required. |
| HTML Slideshow / Instagram Carousel Maker | The output is a horizontal browser presentation. | The output is a portrait social carousel. |
| Digital Lead Magnet Maker / Plain AI Explainer | The resource must earn an opt-in or begin a buyer relationship. | The job is to teach or explain without a conversion asset. |
| Content Performance / Content Refresher | Metrics must be diagnosed. | An existing asset must be updated in the same format. |
| Content Refresher / Content Repurposing | The format and main purpose stay the same. | The source becomes a different format or multiple channel-native assets. |
| Content Repurposing / Instagram Carousel Maker | Several formats or a broad transformation are needed. | The user explicitly asks for one Instagram carousel. |
| Content Repurposing / De-Slop | Format, channel, or asset structure changes. | Only writing style changes. |
| Workflow and SOP Builder / AGENTS.md Setup | A repeated business process is being documented or automated. | The deliverable is an AI instruction file. |
| Big Model Smell / Operator Audit | One supplied artifact, workflow, prompt, or system needs simplification. | Recent local work history is being reviewed with separate explicit approval. |
| Digital Product Launch Builder / Content Repurposing | A coordinated launch sequence, readiness gate, and measurement plan are needed. | Individual source material is being transformed into assets. |

## Universal rules

1. An explicit power name wins unless the requested action is unsafe or impossible.
2. A negative trigger vetoes a result before positive scoring.
3. Exact deliverable and trigger phrases outrank shared topic words.
4. Discovery returns no more than three results; session hints return no more than two.
5. A low-confidence match stays silent rather than forcing a power.
6. When two powers genuinely compose, state which one owns the primary deliverable and use the other only for its distinct part.
