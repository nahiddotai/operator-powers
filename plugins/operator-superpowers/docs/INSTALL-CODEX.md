# Install in Codex and ChatGPT Work

Supported surfaces: ChatGPT Work on the web, ChatGPT Work and Codex in the desktop app, and the Codex CLI plugin browser, wherever your workspace exposes plugins. Ordinary ChatGPT chat, the IDE extension, and mobile are not supported plugin surfaces at the time of writing.

1. Add the `nahiddotai` Git marketplace (this repository) in your plugin settings or plugin browser.
2. Install **Operator Superpowers**.
3. Review the permissions screen: skills, hooks, and one update server (MCP) server.
4. Start a new task. Describe the job naturally, or select the plugin with `@` where available.

Updating: refresh the marketplace and update the plugin. Installed plugins are cached: after a release you may need to update or reinstall and then start a NEW task before the changes appear. This is normal client behaviour, not a broken install.

If your surface shows no plugin support, your workspace plan or client version may not include it yet; Claude Code is the alternative supported route.
